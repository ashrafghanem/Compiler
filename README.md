# Compiler
C code that implements a compiler depending on predefined grammar.

# Description

The code implements the compiler stages as follows:

• File Descriptor

• Scanner

• Symbol Table

• Parser

• Semantic Checker

# Notes: 

- The last stage (Code Generator) has not been implemented yet.
